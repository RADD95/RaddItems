# RaddItems

RaddItems lets you create custom items and item sets through YAML.

Create weapons, armor, consumables, passive hotbar items, and set bonuses with configurable names, lore, enchantments, attributes, equip effects, triggers, and commands.

## Features

- Custom item definitions
- Enchantments and attributes
- Equip-based potion effects
- Trigger-based effects and commands
- Consumable item support
- Item set bonuses
- YAML-based configuration

## Command

```txt
/radditems
```

Aliases:

```txt
/ri
/ci
```

## Permission

```txt
radditems.admin
```

This permission gives access to the plugin administrative command.

## Configuration

RaddItems is built around YAML configuration.

You can define:
- Items
- Attributes
- Equip effects
- Triggers
- Commands
- Sets
- Set bonuses

## Documentation

Detailed configuration examples and guides can be found in the [project wiki](WIKI.MD).

## Example Uses

- Custom swords with hit effects
- Armor with persistent bonuses
- Food with consume effects
- Utility items that work from the hotbar
- Set-based rewards and progression

## Status

RaddItems is currently focused on config-based item creation.  
In-game editors and menus may be added later.
